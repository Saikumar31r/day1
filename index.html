Web Technologies:

HTML

Hypertext Markup Language (HTML) is a standard language for defining the structure and content of web pages.

CSS

Cascading Style Sheets (CSS) is a style sheet language used for specifying the presentation and styling of a document written in a markup language such as HTML ...

Javascript

JavaScript (JS) is a programming language that developers use to create interactive web pages

Frameworks:

React.js

React.js, also known as React, is a free, open-source JavaScript library that helps developers build user interfaces (UIs) for web and native applications.

Bootstrap

Bootstrap is a free, open-source front-end framework that helps developers create websites faster and more easily.

Programming Languages:

Java

Java is a programming language and computing platform that's used to create applications, operating systems, and other software.

Python

Python is a high-level, object-oriented, interpreted programming language that is used for a variety of purposes, including web applications, data science, machine learning, and software development.
